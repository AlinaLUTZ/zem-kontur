# 🏛️ Кадастр.Эксперт  
## Федеральная ИИ-система комплексного анализа земельных участков

> **Автоматизированная правовая и экономическая экспертиза земельных участков в соответствии с ФЗ-218, ЗК РФ и требованиями ФЗ-152**

[![Соответствие ФЗ-152](https://img.shields.io/badge/ФЗ--152-Соответствует-green)](https://digital.gov.ru/ru/documents/6314/)  
[![Суверенный стек](https://img.shields.io/badge/Инфраструктура-Россия-blue)](https://yandex.cloud)

**«Кадастр.Эксперт»** — это production-ready веб-платформа для автоматизированной экспертизы земельных участков, предназначенная для кадастровых инженеров, юридических фирм, банков и образовательных программ. Система анализирует участок по кадастровому номеру и выдаёт:

- Проверку соответствия ВРИ и законодательству  
- Выявление рисков: ЗОУИТ, ЛЭП, водоохранные зоны, наложения границ  
- Прогноз вероятности отказа Росреестра  
- Инвестиционно-экономическую модель освоения  
- Генерацию юридически обоснованных документов  

✅ 100% российская инфраструктура (Yandex Cloud, Selectel)  
✅ Нет персональных данных — только публичные кадастровые номера  
✅ Соответствие ФЗ-152, Приказу ФСТЭК №239, ГОСТ Р 57580.1–2017

## 🛠️ Технологический стек

- **ИИ-ядро**: YandexGPT API (5-го поколения), fine-tuning под земельное право  
- **Backend**: n8n (self-hosted, Docker) + FastAPI (для API)  
- **База данных**: PostgreSQL 15 + PostGIS (геопространственный анализ)  
- **Фронтенд**: React + TypeScript (адаптивный веб-интерфейс)  
- **Инфраструктура**: VPS Selectel (Россия), TLS 1.3, Let’s Encrypt  
- **Безопасность**: `pgcrypto`, IP-фильтрация, аудит действий, автоудаление данных через 30 дней  
- **Интеграции**: ЮKassa, ЭП (Госключ), CRM (Bitrix24, amoCRM)
